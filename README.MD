# OpenModuli
Self-hosted, open-source tool for creating and managing online forms and generating PDFs from them. 
Built with Flask and Python, it offers a user-friendly interface for form design, data collection, and PDF generation. 
Ideal for organizations needing customizable form solutions without relying on third-party services.

## Features
- PDF Export: OpenModuli allows you to export form responses as compact PDF documents, making it easier to share and archive data, or print them.
- Customizable Forms: Create and manage your own forms with a user-friendly builder, without needing to write code.
- Self-Hosted: Unlike cloud-based solutions like Google Forms, you have full control over your data and the application, as it runs on your own server or local machine. No data is stored on third-party servers or sold to advertisers.
- Customizable Branding: Personalize the look and feel of your forms and PDFs with your organization's branding, including colors, logos, and contact information.
- Script Integration: Enhance form functionality with custom Python scripts that can validate data, perform calculations, or integrate with other systems.
- User Management: Create and manage admin users to control access to the form builder and data.
- Open Source: The source code is available on GitHub, allowing for community contributions and custom modifications.

Notes:
- If the username already exists, the command returns an error.
- On first startup, the initial setup wizard can also create or update the admin user.
