pdfutils.create_pdf_from_form_data ( str form_name,
dict form_attributes,
list[dict[str, Any]] nodes,
dict[str, Any] | None submitted_values = None )

Generate a PDF from parsed form nodes and save it to disk.

Args: form_name: Name of the form (e.g., 'iscrizione_corso') form_attributes: Form metadata (title, id, version, lang, submit_label) nodes: Parsed and resolved nodes from parse_fxml

Returns: dict with keys:

  • success (bool): Whether PDF was created successfully
  • pdf_path (str): Path to saved PDF file if successful
  • error (str): Error message if failed
  • pdf_url (str): Download URL path
